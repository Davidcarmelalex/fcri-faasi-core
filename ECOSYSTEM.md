# FAASI-CORE in the Voltex Network

FAASI-CORE is the reliability benchmarking arm of FCRI.

## Position

```
Voltex Network Research Layer
├── FCRI                        # Research institute
│   ├── FAASI Core (this repo)  # Agent reliability benchmark
│   └── AZRAEL                  # Cyber defense
├── MrNothing OS                # Agent platform (benchmark target)
├── VOID//SIGNAL                # Autonomous news (long-horizon tasks)
└── Ayooni                      # Cognitive engine (reasoning tasks)
```

## Benchmark Targets

FAASI-CORE benchmarks evaluate agents from across the Voltex Network:

| Agent System | Benchmark Dimensions |
|-------------|---------------------|
| MrNothing OS | Tool reliability, recovery, stability |
| VOID//SIGNAL | Long-horizon completion, memory integrity |
| Ayooni | Reasoning quality, ambiguity governance |
