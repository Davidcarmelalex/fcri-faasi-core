# Literature Review & Related Work

## Benchmark Landscape

### HELM
HELM focuses on broad language model evaluation across scenarios, emphasizing holistic measurement dimensions.

### SWE-bench
SWE-bench evaluates software engineering capability using real GitHub issue resolution tasks.

### GAIA
GAIA evaluates general AI assistant competence across realistic agentic tasks.

### AgentBench / emerging agent evals
Recent agent benchmarks focus on embodied task completion, tool use, and planning.

## FAASI Positioning
FAASI-CORE specifically targets autonomous operational reliability under long-horizon execution, recovery stress, ambiguity governance, and memory continuity.

## Research Gap
Current benchmarks insufficiently emphasize:
- recovery resilience
- memory integrity
- ambiguity governance
- failure recovery behavior
- long-duration operational reliability
