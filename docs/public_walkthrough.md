# FAASI Public Walkthrough

## What FAASI Is
Benchmark science for autonomous agent reliability.

## Why It Matters
Existing benchmarks insufficiently measure long-horizon operational reliability.

## Demo Flow
1. load benchmark tasks
2. execute benchmark engine
3. score agent outputs
4. export benchmark results
5. summarize performance
