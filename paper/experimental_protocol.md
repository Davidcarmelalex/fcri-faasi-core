# Experimental Protocol

## Benchmark Objective
Evaluate autonomous AI agent reliability across long-horizon tool-augmented workflows.

## Proposed Baseline Systems
- frontier LLM agent runtime
- tool-enabled coding agent
- retrieval-enabled assistant

## Evaluation Categories
- tool use
- filesystem tasks
- recovery tasks
- ambiguity handling
- long-horizon workflow execution

## Evaluation Outputs
- per-task score
- dimension breakdown
- aggregate benchmark score

## Reproducibility Controls
- fixed task definitions
- versioned benchmark inputs
- deterministic scoring logic where applicable
