# FAASI-CORE — Code of Conduct

## Our Pledge

We pledge to make FAASI-CORE a rigorous, reproducible, and collaborative benchmark for autonomous AI agent evaluation.

## Research Standards

### Benchmark Integrity
- All benchmark tasks must be reproducible
- Scoring methodology must be transparent and versioned
- Dataset construction must be documented
- Results must include confidence intervals

### Contribution Standards
- Code must be tested and documented
- New benchmark dimensions require peer review
- Agent evaluation must be fair and unbiased
- All claims must be empirically supported

### Prohibited
- Gaming benchmark scores through overfitting
- Undisclosed use of human intervention
- Plagiarism of benchmark tasks
- Discriminatory behavior in community spaces
